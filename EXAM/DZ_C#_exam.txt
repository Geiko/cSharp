Создать приложение, которое будет предоставлять инормацию о городах и странах.
Содержимое базы стран описываются файлом манифеста. Ниже приведен пример файла с комментариями

//////////////main.manifest//////////////
default_directory=.
editor=notepad.exe {file}
#Обязательные элементы
#Тут ключ - название страны, которое будет отображаться в меню, значение - условное обозначение для дальнейшего использования в манифесте
countries {
	Ukraine=ukr
	United States=us
}
#Тут ключ - страна (из раздела countries), значение - название составной единицы (область, штат) для отображения в меню
distincts {
	ukr=region
	us=state
}
#Ключ - название страны, значение - имя папки в default_directory, в которой находится информация по стране
folders {
	ukr=ukr
	us=us
}
#Элементы, наличие которых определяется из элементов выше
#описание Украины. Tyт, например, dp - имя папки, в которох хранятся файлы с городами Днепропетровской области
ukr {
	Kiev=kiev
	Dnepropetrovsk=dp
}
dp {
	Dnepropetrovsk=dp.txt
	Dneprodzerzhinsk=dpz.txt
}
kiev {
	Kiev=kiev.txt
	Irpen=irp.txt
}

#описание США
us {
	Illinois=il
	Texas=tx
}
il {
	Chikago=chkg.txt
	Avrora=avrora.txt
}
tx {
	Dalas=dls.txt
	Arlington=arl.txt
}
/////////////////////////////////////////////

Необходимо разобрать этот файл и определить файлы и папки в которых хранится информация о городах.
Создать меню, которое будет предлагать пользователю выбрать страну, область (или то, что там в этой стране), 
и город. Программа должна найти файл с текстом о городе, и вывести его на экран. (Например, файл города Днепропетровск находится по пути .\ukr\dp\dp.txt)

* предусмотреть возможность открытия файла (команда открытия нужно прописать в манифесте в строке editor) и внесения туда изменений.
** предусмотреть возможность через интерфейс программы добавлять новые города, области и страны.
